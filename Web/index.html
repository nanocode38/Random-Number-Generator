Features

Everything You Need for
Fair Random Selection

Powerful features wrapped in a beautiful, intuitive interface

CSV Class Rosters

Load student lists from CSV files with flexible formatting. Supports both single-row and dual-row layouts for numbers and names.

Smart Deduplication

Enable deduplication mode to ensure every student is picked fairly. Automatically resets when all students have been selected.

Edge-Hide Mode

Drag the window to screen edges to auto-hide with smooth animations. Hover to bring it back — perfect for presentations.

Light / Dark Themes

Switch between beautiful light and dark themes. The UI adapts instantly with smooth transitions for comfortable use anytime.

EN

Multi-Language

Available in English, 中文简体, 中文繁體, 日本語, and 한국어. More languages can be added via JSON language files.

Cross-Platform

Built with PySide6 (Qt for Python) for native performance on Windows, macOS, and Linux. Lightweight and fast.
